# Pagy

Pagination wrapper for the [pagy](https://github.com/ddnexus/pagy) gem. Displays a "Page X of Y" counter alongside the pagination nav.

## Requirements

The `pagy` gem must be installed in your application.

## Usage

```erb
<%# In your controller %>
@pagy, @items = pagy(Item.all)

<%# In your view %>
<%= jet_ui.pagy(pagy: @pagy) do %>
  <%= pagy_nav(@pagy) %>
<% end %>
```

The component only renders when there is more than one page.

## Options

| Option | Type | Default | Description |
|--------|------|---------|-------------|
| `pagy` | Pagy | required | The Pagy object from the controller |

Any additional keyword arguments are passed through as HTML attributes on the wrapper div.

## CSS classes

The `.pagy` class styles the navigation nav generated by `pagy_nav`. Apply it to any pagination nav element:

| Class | Description |
|-------|-------------|
| `.pagy` | Base class for the pagination nav |
| `.pagy .current` | Current page indicator |
